# Smart Bookmark Agent Codelab

This repository contains `starter_code` and `working_code` code for the [Smart Bookmark Agent]() Codelab.

The `starter_code` directory contains the code for the project with chunks taken out. These chunks are filled in by students as they progress through the lab. The `working_code` directory contains the final working code for the project.

Please refer to the Codelab for further instructions and context.
